📝 Инструкция по работе с таблицами:

  1. Откройте таблицу "Учет возвратов"
  2. Заполните все обязательные поля: дата, артикул, причина, сумма
  3. Калькулятор автоматически рассчитает потери
  4. Используйте фильтры для анализа по периодам и категориям
  5. Еженедельно проверяйте сводку по возвратам

🚨 Критические показатели:

  • Процент возвратов > 15% - требуется срочный анализ
